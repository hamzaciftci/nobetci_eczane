# KVKK Checklist (MVP)

- [x] Konum verisi varsayilan olarak sunucuya gonderilmez.
- [x] Cerez tercihi paneli (opt-in) mevcut.
- [x] Gizlilik/Kullanim/Cerez sayfalari yayinda.
- [x] Yanlis bilgi bildirim akisi mevcut.
- [ ] Veri sorumlusu kimlik/iletisim metni hukuk onayi ile netlestirilecek.
- [ ] Saklama-imha sureleri legal review ile finalize edilecek.
- [ ] Ucuncu taraf servis aktarim analizi (DPA) tamamlanacak.
